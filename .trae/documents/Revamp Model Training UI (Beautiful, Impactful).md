## Scope
- Commit all current changes on `feature/professional-logging-system`
- Exclude `projects/` from the commit
- Push to the same remote branch

## Commands
- `git add -A`
- `git reset projects/`
- `git commit -m "UI revamp: dataset dropdown; GPU device modal; hardware endpoint; config device"`
- `git push origin feature/professional-logging-system`

## Result
- Remote updated with latest UI + backend changes
- `projects/` remains uncommitted
